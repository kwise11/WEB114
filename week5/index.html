What is the purpose of a function?

  1. The purpose of a function is to create code that can be called upon multiple times in your JavaScript. If you have places where you know that the same process needs to be used you can create it one time in the function and then utilize it multiple times. This allows ease of use as you are coding as you do not have to copy and paste or retype the same code and it allows for easy future editing as you can edit the function a single time and the change will be utilized everywhere that function is called.
